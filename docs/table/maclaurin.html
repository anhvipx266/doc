Biến đổi Maclaurin

Hàm số Khai triển chuỗi lũy thừa Bán kính hội tụ \( R \)
\(\frac{1}{1 - x}\) \[ \sum_{n=0}^{\infty} x^n = 1 + x + x^2 + x^3 + \dots \] \(|x| < 1\)
\(\ln(1 + x)\) \[ \sum_{n=1}^{\infty} (-1)^{n+1} \frac{x^n}{n} = x - \frac{x^2}{2} + \frac{x^3}{3} - \dots \] \(|x| \le 1, x \neq -1\)
\(e^x\) \[ \sum_{n=0}^{\infty} \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \dots \] \(R = \infty\)
\(\sin(x)\) \[ \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \dots \] \(R = \infty\)
\(\cos(x)\) \[ \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n}}{(2n)!} = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \dots \] \(R = \infty\)
\(\arctan(x)\) \[ \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{2n+1} = x - \frac{x^3}{3} + \frac{x^5}{5} - \dots \] \(|x| \le 1, x \neq \pm 1\)
\(\frac{1}{1 + x}\) \[ \sum_{n=0}^{\infty} (-1)^n x^n = 1 - x + x^2 - x^3 + \dots \] \(|x| < 1\)
\(\ln(1 - x)\) \[ -\sum_{n=1}^{\infty} \frac{x^n}{n} = -x - \frac{x^2}{2} - \frac{x^3}{3} - \dots \] \(|x| < 1\)
