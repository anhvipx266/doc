Bài tập - Giải tích 3

Phương trình tuyến tính

\(\frac{dy}{dx} + 2y = 6\)

Kết quả: \(y = 3 + Ce^{-2x}\)

Lời giải:

Phương trình có dạng \(\frac{dy}{dx} + P(x)y = Q(x)\), với \(P(x) = 2\) và \(Q(x) = 6\).

Tính tích phân của \(P(x)\):

$$\mu(x) = e^{\int 2 \, dx} = e^{2x}$$

Nhân phương trình với \(\mu(x)\):

$$e^{2x} \frac{dy}{dx} + 2e^{2x} y = 6e^{2x}$$

Phương trình trở thành:

$$\frac{d}{dx} \left( e^{2x} y \right) = 6e^{2x}$$

Tích phân hai vế:

$$e^{2x} y = \int 6e^{2x} dx = 3e^{2x} + C$$

Cuối cùng, ta có:

$$y = 3 + Ce^{-2x}$$

\(y' + 3y = 2x{e^{ - 3x}}\) \[\begin{array}{l}y' + 3y = 2x{e^{ - 3x}}\\p = 3,q = 2x{e^{ - 3x}}\\\rho = {e^{\int {3dx} }} = {e^{3x}}\\{e^{3x}}(y' + 3y) = 2x\\\frac{d}{{dx}}(y{e^{3x}}) = 2x\\y{e^{3x}} = {x^2} + C\\ \Rightarrow y = ({x^2} + C){e^{ - 3x}}\end{array}\]
